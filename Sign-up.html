  • 個人簡介

    　　大學念的是製劑製造工程系，主要是學習藥廠生產製程及品管檢驗方面，畢業退伍後從事醫藥業務近工作近三年的時間，常因一些自己無法控制的阻力而影響我的業績，工作成果需要靠別人決定，讓我感到疲乏，決定轉職學習可以持續累積的專長，所以從2020年4月開始，我透過線上課程學習 HTML, CSS 和JavaScript 等前端工程師需要的語言和技能。
    　　經過一段時間的邊工作邊學習，我期望自己能投入更多時間，專注地學習網頁開發，所以開始著手尋找轉職的培訓資源。2020年中時，報名一間私人公司開設轉職工程師的培訓課程，但由於培訓資格的考題主要是用 Java 進行測試，我大部分時間投入在學習 JavaScript，雖然有成功通過筆試進到面試，但筆試佔據錄取總成績比例大因而沒被錄取。
    　　待業期間我持續的自學，有經濟上的壓力所以也要一邊找工作。目前剛進入淨水器公司維修工程，但我還是不想放棄轉職的任何機會，剛好在堂姐介紹下得知此計畫，課程的內容和技能是轉職成軟體工程師所需要的，因此還是想給自己一個機會投入學習，轉職當工程師。

  • 為什麼想成為網站工程師？

    　　從小我就對電腦感興趣，喜歡玩遊戲，甚至會自己改寫遊戲外掛的腳本，我並不曉得其中已經使用到些微的程式概念，例如條件控制或是迴圈等，當時覺得電腦能夠依照我的指令執行程式是一件非常有趣的事情。從堂姊那得知，包括她在內，有很多非本科系的都成功轉職當工程師，讓我覺得自己應該也有機會可以做到。

  • 為了成為網站工程師，你曾經做過什麼努力？

    　　去年上半年開始利用看Udemy教學影片，學習HTML、CSS以及基本的JavaScript，報名CMONEY主辦的菁英軟體工程師戰鬥營，並學習筆試指定的語言(JAVA)，雖然沒有被錄取，後來持續透過線上課程自學。

  • 對於網站工程師的職業想像是什麼？

    　　只要有任何一個想法，工程師就可以將它實作出來，比如之前的口罩地圖，武漢肺炎病毒的全球分佈人數等等。能夠做到很多事情，產生很大的影響的職業。

  • 如果參與這個計畫，你會怎麼安排學習時間？

    　　像上班上課一樣，規律的作息才能讓自己有足夠時間安排，有效率的管理需要學習的項目，因此計畫每天早上九點開始觀看影片，十點參與會議聆聽當週進度和關鍵技術，中午吃完飯後開始實作當日進度，晚飯時間前休息至晚飯後，如果當日進度尚未完成則繼續，也會找看相關技術的文章或影片來看，並且把學習到的技術整理成筆記。

  • 如果參與這個計畫，你預期會碰到什麼問題？你打算怎麼面對並解決它？

    　　因目前仍有工作，預計會離職來專心學習，但半年沒有收入，可能會有經濟壓力，所以會讓自己力求在計畫結束後快速找到工作，省吃儉用度過這半年。其他預期的困難還有碰到不熟悉的技術，上網找答案、在Slack發問或問堂姐。
